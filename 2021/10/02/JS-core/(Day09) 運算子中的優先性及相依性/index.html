運算子相依性 (Associativity)

這時就要介紹運算子另一個特性，『相依性』，而相依性的特性就是，如果運算子是相同優先性，那麼 JavaScript 會根據設定，從指定方向開始執行，從文件上也可以看到 * 跟 / 的算數運算子是 『從左至右』。
