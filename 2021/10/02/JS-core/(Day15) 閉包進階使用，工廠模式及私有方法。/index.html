私有方法 又稱 模組設計模式（module pattern）

上一章節也有提到，最簡單的閉包就是在函示中 return 一個函示，而私有方法 則是 return 出一個物件，再將各種函示包在物件中，這樣就可以呼叫物件中各式不同的方法，以達成實做時的各種需求，並且和工廠模式相同，使用不同變數、常數閉包儲存的記憶體也會不同，因此可以透過不同參數達成各自獨立計算的需求。
